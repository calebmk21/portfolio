## Project Overview
This was made for ACM Studio’s in-house winter game jam, Snowjam. This was the second game I made with this team, and I still laugh at how this project came about. 

## My Role
I was one of the UI programmers for this project and I learned Unity’s UI Builder feature entirely within the confines of this jam. And by “learned” I mean sort of learned, insofar as the UI works on my Desktop and kind of breaks on most other devices. I still don’t understand how the UI Builder works, nor will I ever claim to, and as cool as I think it could be, I learned afterwards that it is still in alpha and should avoid using it. I pulled an all-nighter trying to debug the volume controller, and it was certainly an interesting experience. I implemented the Main Menu, Settings, Scene Manager (for starting and quitting the game), and the How-To Menu pop ups.

Additionally, I designed the initial concept and gameplay loop for the project, and brainstormed some of the craftable objects.
